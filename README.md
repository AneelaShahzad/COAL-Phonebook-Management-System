# Phonebook Management System (8086 Assembly)

A menu-driven Phonebook Management System developed using **8086 Assembly Language** as part of a Computer Organization and Assembly Language (COAL) semester project.

This project demonstrates low-level programming concepts such as **register manipulation, memory management, array handling, procedures, and interrupts**, while implementing a contact management system.

---

## Features

- Add new contacts
- Search existing contacts
- Display all saved contacts
- Update or edit contact information
- Delete contacts
- Menu-driven console interface
- Efficient use of memory and arrays

---

## Technologies Used

- **8086 Assembly Language**
- **Visual Studio Code** – for writing and editing the code
- **DOSBox** – for compiling and executing the program

---

## How It Works

The program stores contacts in memory using arrays.  
Each contact is managed through a **menu-driven interface**, where the user selects an operation such as adding, searching, updating, or deleting a contact.

Key concepts implemented:

- Register-based operations
- Memory addressing
- Procedures for modular programming
- Interrupts for input/output handling
- Array-based data storage
- Code Reusability

- Implementing structured programming using procedures
- Designing menu-driven programs in a low-level language
