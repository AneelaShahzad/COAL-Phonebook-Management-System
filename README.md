# Phonebook Management System (8086 Assembly)
A menu-driven Phonebook Management System built using 8086 Assembly (COAL).  
This project demonstrates low-level programming concepts like registers, loops, memory handling, arrays, procedures and interrupts, while providing a simple interface to add, search, and manage contacts.

## Key Features
- Add new contacts
- Search existing contacts
- Display all contacts
- Update or Edit Contacts
- Delete any Contact
- Menu-driven interface
- Strong understanding of low-level programming fundamentals

## Technologies Used
- 8086 Assembly Language
- Visual Studios Code (for writing code fastly and efficiently)
- DOSBox (for testing and execution)

## Learning Highlights
- How software interacts with hardware
- Handling memory and registers effectively
- Designing simple yet functional console-based programs
- Memory Management and efficient use of memory
